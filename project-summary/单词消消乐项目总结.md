# 单词消消乐（类似羊了个羊）项目开发总结 - 2026-04-03

## 项目目标
开发一个类似「羊了个羊」的三消单词小游戏，玩法：
- 点击卡片，第一次发音中文，消掉的时候发音英文加鼓励
- 最多8张卡片在卡槽，三张相同自动消除
- 卡槽满了游戏结束
- 统计学会的单词数量，支持分享到微信朋友圈
- 难度递增，每关随机从词库抽卡，支持动态添加新词
