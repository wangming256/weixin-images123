
# 单词消消乐 堆叠式消除学单词微信小游戏
使用 Cocos Creator 开发，完全免费开源

## 项目说明
这是一个**羊了个羊玩法 + 英语单词学习**结合的益智小游戏：
- ✅ 完整的堆叠随机生成算法（模仿羊了个羊）
- ✅ 三张相同卡片自动消除
- ✅ 上层卡片覆盖下层逻辑，只有最上层可点击
- ✅ **点击第一次播中文发音，消除三张后播英文发音**
- ✅ **游戏结束统计：本局你认识了多少单词**，完美寓教于乐
- ✅ **支持一键分享朋友圈**，自带传播
- ✅ 8格卡槽机制，满了就失败
- ✅ 胜利失败弹窗，重新开始
- ✅ **支持预加载所有图片音频**，资源放网上也能一次性加载完不卡顿
- ✅ 适配微信小游戏，一键发布

## 开发环境
1. 下载安装 **Cocos Creator**（免费，支持macOS/Windows）：https://www.cocos.com/creator
2. 用Cocos Creator打开这个项目
3. 配置好你的微信小游戏AppID
4. 点击「项目」→「发布到微信小游戏」，自动导出
5. 在微信开发者工具打开导出的项目，上传审核

## 文件结构
```
assets/
├── scripts/
│   ├── Preloader.ts   # 预加载脚本：启动前一次性加载所有资源 ✨ 新增
│   ├── Game.ts        # 主游戏核心逻辑
│   ├── Card.ts        # 单个卡片组件
│   ├── Slot.ts        # 卡槽组件
│   └── Level.ts       # 关卡生成算法 ✨ 支持单词池随机抽选，每种多张，指数难度
├── textures/           # 放AI生成的卡片纹理
├── audio/              # 放中英文发音和赞美音频
└── scene/
    ├── Preloader.scene # 启动预加载场景
    └── Game.scene     # 主游戏场景
```

## 预加载说明（资源放远程GitHub/jsDelivr时使用）：
如果你把图片和音频放在 GitHub/jsDelivr 上（减小包体积），需要提前一次性加载完：
1. 使用 `Preloader.ts` 做启动场景
2. 提前把所有资源URL添加到预加载列表
3. 显示进度条，全部加载完成再进主游戏
4. 玩的时候完全不卡，体验更好

如果把资源打包在本地，直接跳过预加载也没问题。

## 关卡难度设计
难度指数级增长，完美上瘾设计：

| 关卡 | 单词数量 | 每种单词倍数 | 总卡片数 | 难度 |
|------|----------|--------------|----------|------|
| 1 | 6种 | ×3 → | 18张 | 简单入门 |
| 2 | 12种 | ×6 → | 72张 | 开始变难 |
| 3 | 18种 | ×9 → | 162张 | 很难 |
| 4 | 24种 | ×12 → | 288张 | 地狱难度 |

- 每种单词倍数**始终保证是3的倍数**，永远可以凑对消除
- 卡槽最多8张，总卡片越多越容易满槽失败，越难越想重来
- 默认第二关难度和羊了个羊差不多，一局5-10分钟，完美符合碎片化娱乐

## 美术资源制作
使用免费AI生成足够用了，你只需要：
1. 生成 `n` 种不同图案的卡片（比如水果、动物、图标，每种一张）
2. 尺寸推荐：1024x1024，透明背景PNG
3. 放到 `assets/textures/` 目录
4. 在Cocos编辑器中拖到Card Prefab里就好了

推荐AI提示词：
```
消除游戏卡片，红色圆形苹果，卡通风格，白色背景，透明背景PNG，游戏美术
```

## 玩法说明
和羊了个羊一模一样：
1. 随机堆叠生成关卡，多张卡片叠放有立体效果
2. 点击最上层卡片放到下方卡槽
3. 三张相同卡片自动消除，消除时播放英文发音
4. 消完全部卡片通关，显示本局认识了多少单词
5. 卡槽满了就失败，显示本局认识了多少单词，可以重来
6. 结果页一键分享到朋友圈，自带传播
